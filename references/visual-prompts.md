# 视觉AI提示词规范

> 封面和内文配图的AI生成提示词规范，确保视觉内容与文章调性一致、质量达标。

---

## 一、封面创意策略

### 策略1：冲击型（Impact Style）

**核心逻辑**：用强烈的视觉冲击在0.5秒内抓住注意力。

**适用文章类型**：热点解读、犀利观点、痛点型

**设计要素**：
| 要素 | 要求 |
|------|------|
| 色彩 | 高对比度，主色调1种+强调色1种 |
| 构图 | 主体居中或黄金分割点，留白≤20% |
| 文字 | 大字标题或核心数字，字体粗犷 |
| 情绪 | 紧迫/震撼/警醒 |

**提示词模板**：

```
冲击型封面：
{主题关键词}主题封面图，{主色调}底色，{强调色}点缀，
中心放置{核心视觉元素}，{视觉元素动作/状态}，
{大字内容}文字叠加，粗体{字体风格}字体，
高对比度，视觉冲击力强，
{情绪关键词}氛围，
16:9比例，微信公众号封面尺寸，无水印
```

**示例**：
```
冲击型封面：
AI取代主题封面图，深红底色，白色点缀，
中心放置断裂的代码符号，从中间劈开，
"你的工作还安全吗"文字叠加，粗体无衬线字体，
高对比度，视觉冲击力强，
紧迫警示氛围，
16:9比例，微信公众号封面尺寸，无水印
```

### 策略2：氛围型（Atmosphere Style）

**核心逻辑**：用情绪氛围营造代入感，让读者"感觉到了"。

**适用文章类型**：故事型、情感类、复盘/体验型

**设计要素**：
| 要素 | 要求 |
|------|------|
| 色彩 | 低饱和度，渐变过渡，主色调2种 |
| 构图 | 留白30-40%，主体偏一侧 |
| 文字 | 小字或无字，让氛围说话 |
| 情绪 | 温暖/孤独/宁静/沉思 |

**提示词模板**：

```
氛围型封面：
{情绪关键词}氛围封面图，{色调1}到{色调2}渐变背景，
{场景描述}，{光影描述}，
{核心视觉元素}在{位置}，{元素状态}，
柔焦效果，电影质感，
{情绪关键词}情绪氛围，
16:9比例，微信公众号封面尺寸，无水印
```

**示例**：
```
氛围型封面：
深夜沉思氛围封面图，深蓝到暗紫渐变背景，
城市天际线剪影，远处窗户零星亮光，
一个人影在画面右侧，独自远眺，
柔焦效果，电影质感，
孤独沉思情绪氛围，
16:9比例，微信公众号封面尺寸，无水印
```

### 策略3：信息图型（Infographic Style）

**核心逻辑**：用信息可视化传达核心数据/逻辑，让读者"一眼看懂"。

**适用文章类型**：清单型、对比型、数据分析

**设计要素**：
| 要素 | 要求 |
|------|------|
| 色彩 | 3-4色系统，色块清晰区分 |
| 构图 | 网格/流程/层级结构 |
| 文字 | 数据+标签，信息密度高 |
| 情绪 | 专业/清晰/可信 |

**提示词模板**：

```
信息图型封面：
{主题}信息图封面，{配色方案}配色，
{图表类型}展示{核心数据/逻辑}，
{具体数据点}标注，
{图标元素}点缀，
扁平化设计，清晰专业，
16:9比例，微信公众号封面尺寸，无水印
```

**示例**：
```
信息图型封面：
AI工具对比信息图封面，蓝橙白配色，
雷达图展示3个维度对比，
GPT-4/Claude/Gemini数据标注，
工具图标点缀，
扁平化设计，清晰专业，
16:9比例，微信公众号封面尺寸，无水印
```

## 二、实体锚定规则

### 规则说明

封面中的视觉元素必须是**具体实体**，不能用抽象概念。

| 错误（抽象） | 正确（实体） |
|--------------|-------------|
| "成功" | 奖杯/登顶的人/签下的合同 |
| "焦虑" | 皱眉的人/手机上密密麻麻的通知 |
| "效率" | 计时器/火箭/流程箭头 |
| "创新" | 灯泡(发光)/齿轮(旋转)/拼图(重组) |
| "冲突" | 两股力量碰撞/叉路口/天平倾斜 |

### 实体锚定检查清单

```yaml
entity_anchoring_check:
  - "视觉元素是否为具体可见的物体或场景？"
  - "读者能在0.5秒内识别出元素代表什么？"
  - "元素与文章核心主题是否有明确关联？"
  - "是否避免了抽象符号（如问号、感叹号）作为主元素？"
```

## 三、6种内文配图类型

### 类型1：场景插图（Scene Illustration）

**用途**：为故事/案例提供视觉场景

**结构化提示词模板**：

```yaml
type: scene_illustration
prompt_template: |
  {文章风格}风格插图，{场景描述}，
  {人物动作}，{环境细节}，
  {色调}色调，{情绪}氛围，
  简洁线条/扁平插画风格，
  宽度1080px，适配手机阅读，无水印
usage: "故事型文章的场景还原段"
max_per_article: 2
```

**示例**：
```
温暖风格插图，凌晨的办公室，
一个年轻人对着电脑屏幕揉眼睛，桌上咖啡杯和便利贴，
暖黄色调，孤独但坚持的氛围，
简洁线条插画风格，
宽度1080px，适配手机阅读，无水印
```

## 配图规则

```yaml
image_rules:
  cover:
    required: true
    strategy: auto_select  # 按文章类型自动选择策略
    size: "900x383"        # 微信公众号封面尺寸
    format: "jpg/png"

  inline_images:
    max_per_article: 4
    min_per_article: 1
    size: "1080x_auto"     # 宽度1080px，高度自适应
    format: "jpg/png"
    position_rules:
      - "每800-1200字可插入1张"
      - "不在段落中间断句插入"
      - "金句卡片紧跟金句段落"
      - "数据图表紧跟数据论述"

  style_consistency:
    - "同一篇文章内配图风格统一"
    - "配色方案不超过3种"
    - "图标风格统一（扁平/线条/实色）"
    - "字体风格统一"

  quality_requirements:
    - "分辨率≥72dpi"
    - "无水印"
    - "无版权争议"
    - "适配手机竖屏阅读"
```
