# 7种写作框架

> 微信公众号文章的7种核心写作框架，每种框架包含完整的结构模板和执行策略。

---

## A. 痛点型（Pain Point Framework）

**核心逻辑**：从读者痛点切入，引发共鸣，给出解决方案，建立信任。

### 结构

```
[痛点场景] → [痛点放大] → [原因剖析] → [解决方案] → [行动号召]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 工作效率类 | "每天加班3小时，问题不在能力" |
| 人际关系类 | "为什么你总在讨好别人" |
| 财务焦虑类 | "月薪2万存不下钱，真相扎心了" |
| 职场发展类 | "工作5年还在基层，你缺的不是努力" |
| 健康焦虑类 | "熬夜的人后来都怎么样了" |

### 开头策略

**策略1：场景还原**
> "早上7点的地铁，你挤在人堆里，手机弹出第17条未读消息。"

**策略2：数据冲击**
> "87%的职场人承认：他们每天至少有2小时在做无意义的事。"

**策略3：反常识**
> "你以为你在努力，其实你只是在焦虑。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1-2 | 痛点场景 | 15% | 具体到"就是你"的场景描述 |
| 3-4 | 痛点放大 | 15% | 不解决会怎样？往深了挖 |
| 5-7 | 原因剖析 | 25% | 真正原因≠表面原因，给出洞察 |
| 8-10 | 解决方案 | 30% | 可操作的具体步骤，分点论述 |
| 11-12 | 行动号召 | 15% | 降低行动门槛 + 价值承诺 |

### 金句位置

- **位置1**：痛点放大末尾（"你不是不够努力，是努力的方向错了。"）
- **位置2**：原因剖析核心（"所有行为的底层，都是一种逃避。"）
- **位置3**：解决方案前（"改变不需要从明天开始，从下一个5分钟就行。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 行动型 | 方案可立即执行 | "今天试一个，5分钟就够" |
| 反思型 | 需要思维转变 | "今晚睡前想一个问题" |
| 分享型 | 痛点有强共鸣 | "转给那个总说'我好累'的人" |
| 评论型 | 需要互动数据 | "你中了几条？评论区见" |

## B. 故事型（Story Framework）

**核心逻辑**：通过完整的故事弧线（困境→挣扎→转折→成长），让读者在故事中完成认知转变。

### 结构

```
[日常铺垫] → [意外事件] → [困境挣扎] → [转折点] → [成长领悟] → [读者映射]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 创业故事 | "32岁辞职创业，第一个月我就后悔了" |
| 职场转型 | "从程序员到产品经理，我用了3年" |
| 个人成长 | "那一年我瘦了50斤，但故事不是你想的那样" |
| 人物特写 | "她在工厂做了10年，然后做了一件所有人都觉得疯了的事" |
| 失败复盘 | "我搞砸了一个200万的项目，这是全过程" |

### 开头策略

**策略1：悬念前置**
> "2024年3月15日，我收到了一封改变职业轨迹的邮件。"

**策略2：结果倒叙**
> "现在回头看，那个决定蠢得离谱。但当时，我觉得自己简直是天才。"

**策略3：反差开场**
> "他985毕业，在大厂待了5年，然后去开了一家面馆。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1-2 | 日常铺垫 | 10% | 建立主人公的"正常世界" |
| 3-5 | 意外事件 | 15% | 打破日常，制造张力 |
| 6-9 | 困境挣扎 | 30% | 最长部分，展示真实挣扎而非轻描淡写 |
| 10-11 | 转折点 | 10% | 一个具体的、可信的转折事件 |
| 12-13 | 成长领悟 | 20% | 不是鸡汤，是具体的认知升级 |
| 14 | 读者映射 | 15% | 从"他/她"到"你"的桥接 |

### 金句位置

- **位置1**：意外事件处（"那天我才知道，稳定是一种奢侈。"）
- **位置2**：困境最低点（"最难的不是咬牙，是不知道该不该继续咬。"）
- **位置3**：领悟核心（"成长不是变得更强，是接受了自己的弱。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 共鸣型 | 故事有普遍性 | "你也有过这样的时刻吗？" |
| 勇气型 | 故事涉及重大决策 | "如果重来，你会怎么选？" |
| 期待型 | 系列故事 | "下一篇，讲讲后来怎样了" |
| 分享型 | 故事打动人 | "这个故事值得更多人看到" |

## C. 清单型（List Framework）

**核心逻辑**：通过结构化的信息组织降低认知负担，让读者"学到东西"的获得感极强。

### 结构

```
[场景引入] → [核心观点] → [清单项1] → [清单项2] → ... → [清单项N] → [总结升华]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 方法论 | "高效人士的7个晨间习惯" |
| 资源推荐 | "5个免费AI工具，亲测好用" |
| 避坑指南 | "跳槽前必须想清楚的8件事" |
| 知识盘点 | "2025年必须了解的10个AI概念" |
| 能力清单 | "产品经理的6个核心能力" |

### 开头策略

**策略1：利益承诺**
> "学会这5个技巧，你的效率至少提升一倍。"

**策略2：困惑共情**
> "每次面试被问到'你的职业规划'，你是不是也一脸懵？"

**策略3：权威背书**
> "我面试了200+候选人后总结出的6条铁律。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1 | 场景引入 | 8% | 制造需求，说明为什么这个清单重要 |
| 2 | 核心观点 | 7% | 一句话概括清单背后的逻辑 |
| 3-4 | 清单项1 | 12% | 最重要/最反常识的放第一个 |
| 5-6 | 清单项2 | 12% | |
| ... | ... | ... | |
| N-1到N | 清单项最后 | 12% | 最有行动力的放最后 |
| 倒数2段 | 总结升华 | 12% | 不只总结，要升维 |
| 最后1段 | CTA | 5% | 行动号召 |

**清单项内部结构**：
```
[小标题] → [一句话核心] → [具体说明] → [举例/数据] → [行动建议]
```

### 金句位置

- **位置1**：核心观点段（"清单不是让你做更多，是让你只做对的。"）
- **位置2**：第1个清单项（"最重要的事，永远只有一件。"）
- **位置3**：总结段（"方法千千万，能坚持的才是你的。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 行动型 | 清单可立即执行 | "今天先试第3条，5分钟就够" |
| 收藏型 | 清单适合反复查看 | "先收藏，需要的时候翻出来" |
| 补充型 | 邀请读者补充 | "你还有什么补充？评论区见" |
| 分享型 | 清单有实用价值 | "转给需要的朋友" |

## D. 对比型（Comparison Framework）

**核心逻辑**：通过A/B对比制造认知冲突，让读者在反差中自然接受观点。

### 结构

```
[对比主题引入] → [A面呈现] → [B面呈现] → [差距放大] → [深层原因] → [选择指引]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 认知对比 | "月薪1万和月薪10万的人，差的不只是钱" |
| 方法对比 | "学习用这2种方式，3个月后差距惊人" |
| 人生选择 | "留在大城市还是回老家？5年后的差距" |
| 代际对比 | "90后和00后的消费观，完全不是一个物种" |
| 中外对比 | "中美AI差距，没你想的那么大" |

### 开头策略

**策略1：两组数据**
> "同一个岗位，A公司开出3万，B公司只给1.5万。差在哪？"

**策略2：两种人生**
> "同样是35岁，有人年薪百万，有人还在投简历。"

**策略3：认知反转**
> "你以为的劣势，可能是最大的优势。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1-2 | 对比主题引入 | 10% | 明确对比对象和核心差异点 |
| 3-5 | A面呈现 | 20% | 具体细节，不是概括 |
| 6-8 | B面呈现 | 20% | 同等篇幅，细节对等 |
| 9-10 | 差距放大 | 15% | 量化差距，制造冲击 |
| 11-13 | 深层原因 | 25% | 这是全文价值核心——为什么不同 |
| 14-15 | 选择指引 | 10% | 给出具体建议 |

### 金句位置

- **位置1**：对比引入（"选择比努力重要，但多数人选错了方向还在拼命跑。"）
- **位置2**：差距放大（"差距不是一天拉开的，是每天0.1%的复利。"）
- **位置3**：深层原因（"表面看是方法的差距，底层是思维的差距。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 选择型 | 文章帮读者做选择 | "你现在在A还是B？" |
| 反思型 | 需要思维转变 | "你属于哪一种？" |
| 行动型 | 有明确行动方向 | "从今天起，试试B方案" |
| 分享型 | 对比有争议性 | "你认同吗？转给朋友聊聊" |

## E. 热点解读型（Trending Analysis Framework）

**核心逻辑**：快速响应热点事件，提供超越情绪的深度解读，给读者"懂了"的获得感。

### 结构

```
[热点事件] → [大众反应] → [被忽略的真相] → [深层逻辑] → [未来影响] → [读者行动]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 行业事件 | "OpenAI发布GPT-5，真正值得关注的3件事" |
| 政策变化 | "新个税法实施，这3类人影响最大" |
| 突发新闻 | "XX裁员5000人，背后是整个行业的转型" |
| 社会现象 | "考公人数再创新高，说明了什么" |
| 技术突破 | "量子计算新突破，离你有多远" |

### 开头策略

**策略1：事件概述+悬念**
> "昨天，XX公司宣布了一件事。多数人在讨论对错，但真正重要的，是没人注意的那一点。"

**策略2：反大众立场**
> "大家都在骂，但我看到了一个不一样的角度。"

**策略3：影响预判**
> "这件事会改变3个行业的格局，但90%的人还没意识到。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1-2 | 热点事件 | 10% | 简洁概述，不展开细节 |
| 3-4 | 大众反应 | 10% | 描述主流观点，为反转蓄势 |
| 5-7 | 被忽略的真相 | 25% | 核心价值——你看到的别人没看到的 |
| 8-10 | 深层逻辑 | 25% | 为什么会这样？结构性的原因 |
| 11-12 | 未来影响 | 15% | 对读者的具体影响 |
| 13-14 | 读者行动 | 15% | 可操作的建议 |

### 金句位置

- **位置1**：大众反应后（"热闹是别人的，但影响是所有人的。"）
- **位置2**：真相揭示（"表面是A的问题，本质是B的危机。"）
- **位置3**：未来影响（"今天不思考，明天就被安排。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 预判型 | 邀请读者预测 | "你觉得3个月后这事会怎样？" |
| 行动型 | 有明确行动建议 | "接下来这3件事，建议你现在就做" |
| 分享型 | 热点有传播性 | "别只看热闹，转给该看的人" |
| 讨论型 | 观点有争议 | "你同意吗？来评论区聊聊" |

## F. 纯观点型（Opinion Framework）

**核心逻辑**：直接输出鲜明观点，用逻辑和论据支撑，不绕弯子。适合已建立信任的作者。

### 结构

```
[观点直出] → [核心论证] → [反面观点驳斥] → [补充论据] → [观点深化] → [结语]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 行业判断 | "AI不会取代程序员，但会改变这个职业" |
| 价值观输出 | "内卷不是问题，躺平才是" |
| 趋势预判 | "2025年，这3个行业会爆发" |
| 反常识观点 | "副业正在毁掉你的主业" |
| 经验总结 | "做了10年管理，我最后悔的3件事" |

### 开头策略

**策略1：观点直陈**
> "我不装了，先说结论：副业90%是浪费时间。"

**策略2：挑战常识**
> "所有人都说'要坚持'，但我想说：该放弃的时候就放弃。"

**策略3：身份背书**
> "在行业摸爬滚打15年，我看到了一个很多人不愿承认的真相。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1 | 观点直出 | 8% | 不超过3句话，观点必须鲜明 |
| 2-4 | 核心论证 | 25% | 3个论据，从强到更强 |
| 5-7 | 反面驳斥 | 20% | 预判反对意见并逐一反驳 |
| 8-9 | 补充论据 | 15% | 数据/案例/权威引用 |
| 10-11 | 观点深化 | 20% | 从"是什么"到"为什么重要" |
| 12 | 结语 | 12% | 不重复观点，升维或留白 |

### 金句位置

- **位置1**：观点直出处（"不是所有坚持都叫坚持，有些叫固执。"）
- **位置2**：反驳后（"反对的声音越大，说明踩到了痛点。"）
- **位置3**：深化处（"这不是一个选择的问题，是一个认知的问题。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 站队型 | 观点鲜明有争议 | "你站哪边？评论区投票" |
| 反驳型 | 鼓励不同声音 | "不同意？来辩" |
| 思考型 | 需要深度思考 | "想一想，你真的同意主流观点吗？" |
| 分享型 | 观点有启发性 | "值得和在乎的人聊聊" |

## G. 复盘/体验型（Review/Experience Framework）

**核心逻辑**：以第一人称真实经历为基础，展示过程中的得失和思考，让读者"跟着走了一遍"。

### 结构

```
[背景交代] → [目标设定] → [执行过程] → [结果呈现] → [得失分析] → [可复用经验]
```

### 适用场景

| 场景 | 示例主题 |
|------|----------|
| 项目复盘 | "我做了一个月小红书，数据全公开" |
| 学习体验 | "花3万报了个培训班，值不值" |
| 产品体验 | "用了30天ChatGPT Plus，说说真实感受" |
| 方法测试 | "每天5点起床30天，我发现了3个真相" |
| 失败复盘 | "投入10万做副业，3个月亏完了" |

### 开头策略

**策略1：结果先行**
> "先说结果：30天涨粉2万，但花了1.8万。值不值？往下看。"

**策略2：时间锚定**
> "30天前我决定做一件事。今天，是第30天。"

**策略3：质疑切入**
> "有人说这事很简单。我来告诉你，简单个鬼。"

### 段落结构

| 段落序号 | 功能 | 字数占比 | 要点 |
|----------|------|----------|------|
| 1-2 | 背景交代 | 10% | 为什么做这个？什么条件下做的？ |
| 3 | 目标设定 | 8% | 具体目标（可量化的） |
| 4-8 | 执行过程 | 30% | 按时间线或关键节点展开，保留真实细节 |
| 9-10 | 结果呈现 | 12% | 数据说话，好和坏都说 |
| 11-12 | 得失分析 | 20% | 哪些做对了，哪些做错了 |
| 13-14 | 可复用经验 | 20% | 读者能直接用的 |

### 金句位置

- **位置1**：执行过程的困难点（"别人只看到结果，没人关心你凌晨3点还在调参数。"）
- **位置2**：得失分析（"最大的收获不是2万粉丝，是搞清楚了一件事：我不适合做内容。"）
- **位置3**：经验总结（"复盘不是为了后悔，是为了下次少走弯路。"）

### CTA策略

| CTA类型 | 适用场景 | 示例 |
|---------|----------|------|
| 提问型 | 邀请读者分享类似经历 | "你做过类似的事吗？结果怎样？" |
| 请教型 | 寻求更好方法 | "有经验的朋友，评论区指教" |
| 追踪型 | 持续更新 | "关注我，下个月再更新" |
| 分享型 | 经验有参考价值 | "转给也在做这件事的人" |
