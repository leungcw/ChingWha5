# 首次设置引导

> 4阶段交互式设置流程，帮助新用户完成账号配置和初始化。

---

## 阶段概览

```
阶段1：收集信息 → 阶段2：生成配置 → 阶段3：可选语料生成playbook → 阶段4：试运行
```

---

## 阶段1：收集信息

### 1.1 基本信息

| 收集项 | 说明 | 示例 |
|--------|------|------|
| 公众号名称 | 账号名称 | "AI工具测评站" |
| 账号定位 | 一句话描述 | "专注AI工具的实测和推荐" |
| 目标受众 | 读者画像 | "对AI工具感兴趣的职场人，25-40岁" |
| 更新频率 | 发布节奏 | "每周2-3篇" |

**交互方式**：逐项提问，每次1个问题

```
🤖 欢迎使用微信公众号写作助手！我们来快速完成设置。

第一个问题：你的公众号叫什么名字？
> AI工具测评站

收到！接下来：请用一句话描述你的公众号定位。
> 专注AI工具的实测和推荐

好的。你的目标读者是谁？（比如：年龄、职业、兴趣）
> 对AI工具感兴趣的职场人，25-40岁

了解。你计划多久发一篇文章？
> 每周2-3篇
```

### 1.2 写作偏好

| 收集项 | 选项 | 说明 |
|--------|------|------|
| 喜欢的写作风格 | 犀利/温暖/专业/幽默/冷静 | 选1个主风格 |
| 文章长度偏好 | 短篇(800-1500)/中篇(1500-2500)/长篇(2500+) | 选1个 |
| 是否接受口语化 | 是/否 | 影响词汇温度 |
| 核心话题领域 | 最多选3个 | 影响选题范围 |

**交互方式**：

```
接下来了解你的写作偏好：

你更喜欢哪种写作风格？
  1. 犀利直接（一针见血，不留情面）
  2. 温暖贴心（像朋友聊天，有温度）
  3. 专业权威（像行业专家，有深度）
  4. 幽默风趣（会讲故事，有笑点）
  5. 冷静理性（用数据说话，不煽情）
> 3

你偏好哪种文章长度？
  1. 短篇（800-1500字，快读干货）
  2. 中篇（1500-2500字，有深度）
  3. 长篇（2500+字，深度长文）
> 2

你的文章可以接受口语化表达吗？（比如"炸裂""绝了"这种词）
> 可以，但别太多

你的核心话题领域是？（最多3个）
  1. AI/科技
  2. 职场/效率
  3. 财务/理财
  4. 成长/学习
  5. 生活/情感
  6. 行业/商业
  7. 其他（请说明）
> 1, 2, 6
```

### 1.3 人格匹配

基于阶段1.1和1.2的回答，推荐最匹配的人格：

```yaml
persona_matching:
  rules:
    - style: "犀利" → sharp-journalist
    - style: "温暖" → midnight-friend
    - style: "专业" → warm-editor
    - style: "幽默" → humor-storyteller
    - style: "冷静" → cold-analyst

  topic_adjustment:
    - topics_include: ["AI/科技"] → tech-coder 候选
    - topics_include: ["行业/商业"] → industry-observer 候选
```

**交互方式**：

```
根据你的偏好，我推荐你使用【温暖主编 warm-editor】人格：
  专业有温度，像一个靠谱的前辈在给你支招。

  不过，考虑到你写AI/科技话题，【极客码农 tech-coder】也很适合：
  硬核实操，像技术大佬在教你。

你想用哪个？
  1. 温暖主编（推荐）
  2. 极客码农
  3. 两个都想看看效果
> 1
```

## 阶段3：可选语料生成playbook

### 3.1 语料来源选择

```
接下来可以帮你建立"个人写作风格库"。

你有现成的公众号文章吗？
  1. 有，我可以提供链接/文件
  2. 有，但暂时不方便提供
  3. 没有，这是新号

（这一步是可选的，跳过也能正常使用）
> 1
```

### 3.2 语料导入

```yaml
corpus_import:
  sources:
    - type: "url"
      description: "提供公众号文章链接"
      process: "抓取→清洗→分析"

    - type: "file"
      formats: ["txt", "md", "docx", "html"]
      description: "上传文件"
      process: "解析→清洗→分析"

    - type: "paste"
      description: "直接粘贴文章内容"
      process: "清洗→分析"
```

### 3.3 语料分析

从导入的文章中提取写作模式：

```yaml
corpus_analysis:
  min_articles: 3
  max_articles: 20

  analysis_dimensions:
    - vocabulary_patterns:     # 词汇偏好
        - "常用口头禅/固定表达"
        - "偏好句式开头"
        - "不使用的词汇"

    - structure_patterns:      # 结构偏好
        - "平均段落长度"
        - "独立段使用频率"
        - "开头模式偏好"
        - "结尾模式偏好"

    - emotion_patterns:        # 情绪偏好
        - "情绪强度分布"
        - "情绪转折频率"

    - persona_patterns:        # 人格倾向
        - "主导人格特征"
        - "人格漂移模式"
```

### 3.4 playbook生成

```
我分析了你提供的5篇文章，提取了以下写作特征：

📊 你的写作特征
  - 平均段落长度：68字（偏短段）
  - 喜欢用反问句（每篇平均3.2个）
  - 开头偏好：场景还原式
  - 结尾偏好：行动号召式
  - 口头禅："说实话""你猜怎么着"
  - 不使用的词："综上所述""值得注意的是"

这些特征已保存到你的个人playbook，后续写作会参考这些模式。
```

## 阶段4：试运行

### 4.1 试写选题

```

  2. ChatGPT和Claude到底选哪个？实测对比（对比型，预估82分）
  3. 我用AI写了一个月的公众号，数据全公开（复盘型，预估75分）

你想试哪个？或者自己提一个选题？
> 2
```

### 4.2 试写过程

```

  框架：D对比型
  人格：温暖主编
  增强策略：density_boost + detail_anchoring
  预估长度：2000字

开始写作中...
```

### 4.3 试写反馈

```

  1. 整体感觉如何？（1-5分）

收到！我会调整词汇温度到0.72，增加口语化表达。
```

      name: "生成配置"
