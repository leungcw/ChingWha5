# 非管道命令调度表

> 所有非管道命令的触发词和动作映射，供 Skill 识别和调度。

---

## 命令分类

| 类别 | 命令数 | 说明 |
|------|--------|------|
| 配置命令 | 5 | 修改账号/写作配置 |
| 选题命令 | 3 | 选题相关操作 |
| 学习命令 | 3 | 学习飞轮相关 |
| 效果命令 | 3 | 效果复盘相关 |
| 工具命令 | 4 | 辅助工具 |
| 紧急命令 | 2 | 紧急处理 |

---

## 配置命令

### /setup

| 属性 | 说明 |
|------|------|
| 触发词 | `/setup`, `设置`, `初始化`, `首次设置` |
| 动作 | 启动4阶段交互式设置引导 |
| 参数 | 无 |
| 前置条件 | 无（新用户或重置配置） |
| 输出 | 生成 style.yaml 和 writing-config |

### /config

| 属性 | 说明 |
|------|------|
| 触发词 | `/config`, `配置`, `修改配置`, `调整配置` |
| 动作 | 修改指定配置项 |
| 参数 | `key=value` 格式，如 `/config 词汇温度=0.75` |
| 可修改项 | vocabulary_temperature, sentence_length_variance, emotional_polarity_range, adverb_density_max, preferred_length, persona |
| 输出 | 确认修改并更新配置文件 |

### /persona

| 属性 | 说明 |
|------|------|
| 触发词 | `/persona`, `人格`, `换人格`, `切换风格` |
| 动作 | 切换或查看写作人格 |
| 参数 | 人格名称或"list" |
| 示例 | `/persona sharp-journalist`, `/persona list` |
| 输出 | 人格切换确认或人格列表 |

### /style

| 属性 | 说明 |
|------|------|
| 触发词 | `/style`, `风格`, `查看风格` |
| 动作 | 查看当前风格配置 |
| 参数 | 无（查看全部）或指定字段名 |
| 输出 | 当前 style.yaml 内容摘要 |

### /reset

| 属性 | 说明 |
|------|------|
| 触发词 | `/reset`, `重置`, `恢复默认` |
| 动作 | 重置指定配置到默认值 |
| 参数 | "all" 或指定配置项 |
| 前置条件 | 需确认（重置不可逆） |
| 输出 | 确认重置 |

## 选题命令

### /topic

| 属性 | 说明 |
|------|------|
| 触发词 | `/topic`, `选题`, `推荐选题`, `换个话题` |
| 动作 | 推荐3个候选选题 |
| 参数 | 可选：指定领域或关键词 |
| 示例 | `/topic AI工具`, `/topic` |
| 流程 | 3维度评估 → 去重检查 → 排序推荐 |
| 输出 | 3个选题及评分 |

### /evaluate

| 属性 | 说明 |
|------|------|
| 触发词 | `/evaluate`, `评估`, `评估选题`, `这个选题怎么样` |
| 动作 | 评估指定选题 |
| 参数 | 选题描述 |
| 示例 | `/evaluate AI工具推荐清单` |
| 流程 | 3维度评分 → 去重检查 → 框架推荐 |
| 输出 | 评分报告 + 框架建议 |

### /history

| 属性 | 说明 |
|------|------|
| 触发词 | `/history`, `历史`, `选题历史`, `写过什么` |
| 动作 | 查看选题历史 |
| 参数 | 可选：时间范围（7天/30天/全部） |
| 输出 | 历史选题列表及状态 |

## 学习命令

### /learn

| 属性 | 说明 |
|------|------|
| 触发词 | `/learn`, `学习`, `提交修改`, `这是终稿` |
| 动作 | 提交初稿+终稿对，触发学习飞轮 |
| 参数 | 终稿内容（初稿已有记录） |
| 流程 | diff分析 → 模式提取 → 置信度评分 |
| 输出 | 学习结果摘要 |

### /playbook

| 属性 | 说明 |
|------|------|
| 触发词 | `/playbook`, `写作手册`, `查看规则`, `我的偏好` |
| 动作 | 查看当前playbook |
| 参数 | 无（查看全部）或指定类别 |
| 示例 | `/playbook vocabulary` |
| 输出 | playbook 内容摘要 |

### /import

| 属性 | 说明 |
|------|------|
| 触发词 | `/import`, `导入`, `导入文章`, `导入语料` |
| 动作 | 导入历史文章用于学习 |
| 参数 | URL/文件路径/粘贴内容 |
| 流程 | 解析 → 清洗 → 分析 → 模式提取 |
| 输出 | 提取结果摘要 + playbook更新 |

## 工具命令

### /check

| 属性 | 说明 |
|------|------|
| 触发词 | `/check`, `检查`, `自检`, `AI检测` |
| 动作 | 对指定文本执行反AI检测自检 |
| 参数 | 文本内容或"latest"（最近生成的文章） |
| 流程 | 统计层检查 → 语言层检查 → 内容层检查 |
| 输出 | 自检报告 + 问题标注 |

### /seo

| 属性 | 说明 |
|------|------|
| 触发词 | `/seo`, `SEO检查`, `SEO优化` |
| 动作 | 对指定文本执行SEO检查 |
| 参数 | 文本内容或"latest" |
| 流程 | 标题→摘要→关键词→标签→完读率 |
| 输出 | SEO检查报告 + 优化建议 |

### /outline

| 属性 | 说明 |
|------|------|
| 触发词 | `/outline`, `大纲`, `生成大纲`, `先看大纲` |
| 动作 | 根据选题和框架生成文章大纲 |
| 参数 | 选题 + 可选框架类型 |
| 输出 | 结构化大纲 |

### /rewrite

| 属性 | 说明 |
|------|------|
| 触发词 | `/rewrite`, `重写`, `改写`, `这段重写` |
| 动作 | 重写指定段落 |
| 参数 | 段落内容 + 可选重写方向 |
| 示例 | `/rewrite 更口语一点`, `/rewrite 加点数据` |
| 输出 | 重写后的段落 |

## 紧急命令

### /stop

| 属性 | 说明 |
|------|------|
| 触发词 | `/stop`, `停`, `停止`, `暂停` |
| 动作 | 立即停止当前写作/生成任务 |
| 参数 | 无 |
| 优先级 | 最高 |
| 输出 | 停止确认 + 已完成部分保存 |

### /abort

| 属性 | 说明 |
|------|------|
| 触发词 | `/abort`, `放弃`, `作废`, `不要了` |
| 动作 | 放弃当前文章，不保存 |
| 参数 | 无 |
| 前置条件 | 需二次确认 |
| 输出 | 放弃确认 |

## 命令调度逻辑

```yaml
command_dispatcher:
  recognition:
    - "精确匹配触发词"
    - "模糊匹配（包含触发词的句子）"
    - "上下文推断（如'帮我看看SEO'→/seo）"

  priority:
    emergency: 100    # /stop, /abort
    config: 80        # /config, /persona, /reset
    action: 60        # /topic, /check, /seo
    learning: 40      # /learn, /import
    info: 20          # /style, /history, /playbook, /stats

  pipeline_vs_command:
    pipeline: "写作管道（选题→大纲→写作→自检→SEO→输出）"
    command: "独立命令（上述所有命令）"
    rule: "管道运行中，紧急命令立即执行，其他命令排队"
