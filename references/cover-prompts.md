## 封面生成配置

```yaml
cover_generation:
  auto_select: true          # 自动根据框架类型选择封面策略
  fallback: "atmosphere"     # 默认使用氛围型

  strategy_mapping:
    A_pain_point: "impact"
    B_story: "atmosphere"
    C_list: "infographic"
    D_comparison: "impact"
    E_trending: "impact"
    F_opinion: "impact"
    G_review: "atmosphere"

  size: "900x383"
  format: "jpg"
  quality: 90

  mandatory_elements:
    - "主题相关的具体视觉元素（非抽象）"
    - "文字不超过10个汉字"
    - "无水印"
    - "品牌色出现至少1次"

  forbidden_elements:
    - "纯文字封面（无视觉元素）"
    - "过于复杂的场景"
    - "小字过多（手机看不清）"
    - "纯白/纯黑底无元素"
```
