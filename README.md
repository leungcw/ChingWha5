# 宇宙第一公众号全自动写作助手

> 跟 AI 说一句话，它帮你把选题、写稿、排版、配图全干了。你只需要复制粘贴到公众号后台。

---

## 🤔 一句话说清楚

你平时写一篇公众号文章要多久？选题 → 查资料 → 写稿 → 改稿 → 排版 → 配图 → 预览 → 发布。

现在你只需要说：

> "帮我写一篇关于 XX 的公众号文章"

剩下的它全包了。**8分钟出一篇能直接发的文章。**

---

## 🎯 能干啥

### ① 一键出文（最常用）

说个主题，整篇文章直接出来——选题分析、文章框架、正文写作、去 AI 味、配图、排版、预览，一步到位。

### ② 模仿你的风格写

给它看几篇你以前写的文章，它就能学你的语气、你的节奏、你的口头禅。不是模板套壳，是真正像你写的。

### ③ 排版不翻车

所有排版都是纯内联样式，复制到微信公众号后台不会乱。你看到的预览，就是发出去的样子。

## 🚀 2 步上手（不需要会任何技术）

这个东西的本质是一套"写给 AI 看的说明书"——告诉 AI 怎么写公众号文章、怎么排版、怎么去 AI 味。你只需要把这份说明书塞给你常用的 AI 就行。

### 第 1 步：塞给 AI（选一个你在用的）

| 你在用什么 AI | 怎么塞 |
|---------------|--------|
| **ChatGPT** | 新建一个对话，把 `SKILL.md` 文件里的内容全部复制粘贴进去，说"按这个规则帮我写文章"。或者创建 Custom GPT，把整个文件夹传上去当知识库 |
| **Claude** | 创建 Project，把所有文件丢进 Project 知识库。或者直接粘贴 SKILL.md 到对话开头 |
| **CodeBuddy** | 说"帮我安装一个 Skill"，它会一步步教你。1 分钟搞定 |
| **Codex CLI** | 把 `SKILL.md` 和 `references/` 文件夹放到项目里，或者在对话里粘贴 SKILL.md 内容 |
| **其他 AI** | 都一样——把 `SKILL.md` 的内容粘到对话开头，说"按这个来"就行 |

本质上就是把 SKILL.md 这个文件的内容给你的 AI 读一遍。不管用什么工具，道理一样。

### 第 2 步：用

塞好之后，直接跟它说话。不需要记任何命令。

## 📋 实际走一遍

假设你想写一篇关于"年轻人被骗"的文章。

**你：** "帮我写一篇关于年轻人被骗的公众号文章，受众 23 岁左右，口语一点，不要太说教"

**它会：**
1. 先分析这个选题好不好
2. 给你看文章框架（你点头它才继续）
3. 写正文
4. 去 AI 痕迹（三层检测，确保读起来不像 AI 写的）
5. 高级排版（带章节数字、金句高亮、清爽留白）
6. 生成预览页面

**你：** 打开预览 → 看着满意 → 全选复制 → 贴到公众号后台 → 发了。

全程你只动了两次手：确认框架 + 复制粘贴。

## 🙋 常见问题

**Q：我不会写代码，能用吗？**
能。你不需要会任何代码。不管用的 ChatGPT、Claude 还是别的 AI，把 SKILL.md 粘进去就能用。

**Q：装起来麻烦吗？**
不麻烦。本质就是拷贝粘贴——把 SKILL.md 的内容粘给你的 AI，说"按这个来"。不论什么平台，最多 1 分钟。

**Q：写出来的文章有 AI 味吗？**
没有。它有三层去 AI 检测，而且能学你的写作风格。你给它看的你的文章越多，它写得越像你。

**Q：排好版复制到公众号会不会乱？**
不会。所有样式都是微信能识别的格式，复制过去就是看到的那个样子。

**Q：要不要钱？**
工具本身完全免费、开源。配图如果用 AI 生成，可能会有少量 API 费用。

**Q：可以把以前的文章给它学吗？**
可以。把文章链接或文字发给它，说"学一下这个风格"就行。

**Q：能自动发到公众号吗？**
可以。配置好微信 API 凭证后，它能直接帮你发布到草稿箱。不配也行，手动复制更安全。
